
This is a modern website for corporate organizations implementing a simple flask server, a cleaning service business, built with Flask and Bootstrap.

## Features
- Homepage
- About page
- Services page
- Contact page
- Booking form

## Setup
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the app:
   ```bash
   python app.py
   ```
3. Visit [http://localhost:5000](http://localhost:5000) in your browser.

## Notes
- All images are placeholders. Replace with your own assets as needed.
- Booking and contact forms are for demonstration and do not send emails or store data.
